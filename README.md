# ⏰ Relógio Digital

Um relógio digital moderno, bonito e totalmente responsivo feito com **HTML5, CSS3 e JavaScript puro**.

Projeto desenvolvido com o objetivo de praticar manipulação de tempo, animações CSS e criação de interfaces clean.

## ✨ Funcionalidades

- Exibição em tempo real de horas, minutos e segundos
- Formato 24 horas
- Atualização automática a cada segundo
- Design minimalista e moderno
- Totalmente responsivo (funciona bem em celular, tablet e desktop)
- Animações suaves com CSS

## 🛠️ Tecnologias utilizadas

- **HTML5** — Estrutura semântica
- **CSS3** — Estilização + animações
- **JavaScript (Vanilla)** — Manipulação do tempo com `Date()`
